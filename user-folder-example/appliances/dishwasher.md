# Dishwasher - Model DW-5500

## Product Information
**Brand:** CleanPro  
**Model:** DW-5500  
**Purchase Date:** January 10, 2023  
**Warranty:** 5 years parts and labor

## Features
- Energy Star rated
- 6 wash cycles (Normal, Heavy, Quick, Eco, Rinse, Sanitize)
- Adjustable upper rack
- Stainless steel interior
- Quiet operation (45 dB)
- Delay start timer

## Operating Instructions
1. Load dishes properly (plates in bottom rack, glasses on top)
2. Add detergent to dispenser
3. Add rinse aid if needed
4. Select wash cycle
5. Press START
6. Wait for cycle to complete (45 min - 2.5 hours depending on cycle)

## Loading Tips
- Place large items on sides and back
- Angle bowls and cups downward
- Don't nest items together
- Utensils should face up (except knives)
- Remove food debris before loading

## Maintenance
- **Weekly:** Wipe door seals
- **Monthly:** Clean filter (bottom of dishwasher)
- **Quarterly:** Run cleaning cycle with dishwasher cleaner
- **As needed:** Refill rinse aid dispenser

## Cycles Guide
- **Normal (1.5 hrs):** Everyday dishes
- **Heavy (2.5 hrs):** Pots, pans, heavily soiled items
- **Quick (45 min):** Lightly soiled dishes
- **Eco (2 hrs):** Energy-saving option
- **Rinse:** Rinse only, no detergent
- **Sanitize (2 hrs):** High-temperature final rinse

## Troubleshooting
- **Not draining:** Check filter for blockage
- **Dishes not clean:** Use more detergent, clean spray arms
- **White film on dishes:** Add rinse aid, check water hardness
- **Noisy operation:** Check for items blocking spray arms
