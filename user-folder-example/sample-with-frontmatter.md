# Sample Note with Frontmatter

This is a sample note that demonstrates the new YAML frontmatter support in Noteel.

- Tags: example, tutorial, markdown

You can now organize your notes with metadata that makes them easier to find and filter!

## How to Use

1. Add YAML frontmatter at the top of your note
2. Use the metadata fields when editing a note
3. Filter notes by tags or categories using the dropdowns in the search bar
4. See match counts per folder when searching

Enjoy the new features!
